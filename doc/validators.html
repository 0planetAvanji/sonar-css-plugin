Browsers ignore invalid property values. Review those invalid property values and either remove them if they are useless or update them.

Noncompliant Code Example

/* 'word-spacing' validator is 'normal | <length>' */
.mybox {
  word-spacing: 10px
  word-spacing: normal
  word-spacing: 0
  word-spacing: inherit
  word-spacing: 10       /* Noncompliant: Missing length unit */
  word-spacing: abc      /* Noncompliant: 'abc' value is not allowed */
}
