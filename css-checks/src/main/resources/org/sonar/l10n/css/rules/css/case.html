Even though CSS syntax is case-insensitive, sharing some style conventions is a key point to make it possible for a team to efficiently collaborate.

This rule allows to check that all properties and functions are written in lowercase.

Noncompliant Code Example

.mybox {
  color: blue;
  COLOR: blue; /* Noncompliant */
  Color: blue; /* Noncompliant */;
}

.mybox {
  color: rgb(255,255,255;
  color: RGB(255,255,255; /* Noncompliant */
  color: Rgb(255,255,255; /* Noncompliant */;
}

Compliant Solution

.mybox {
  color: blue;
}

.mybox {
  color: rgb(255,255,255;
}
