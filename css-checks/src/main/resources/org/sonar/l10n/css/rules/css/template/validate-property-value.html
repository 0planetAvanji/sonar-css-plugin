Browsers ignore invalid property values. Review those invalid property values and either remove them or update them.

Noncompliant Code Example

/* 'word-spacing' validator is 'normal | <length>' */
.mybox {
  word-spacing: 10px;
  word-spacing: normal;
  word-spacing: 0;
  word-spacing: inherit;
  word-spacing: 10;       /* Noncompliant: Missing length unit */
  word-spacing: abc;      /* Noncompliant: 'abc' value is not allowed */
}

Implemented Validators

Based on W3C CSS specifications Level 1 or 2 or 3 which status is either Completed or Testing or Refining. Note that not all validators are implemented for now.
