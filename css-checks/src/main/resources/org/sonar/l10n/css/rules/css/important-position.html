The !important statement must be placed at the end of the line, immediately before the semicolon, otherwise it will not be taken into account by the browser.

Noncompliant Code Example

.mybox {
  width: !important 100px;             /* Noncompliant */
  border: 1px solid !important black;  /* Noncompliant */
}

Compliant Solution

.mybox {
  width: 100px !important;             /* Compliant */
  border: 1px solid black !important;  /* Compliant */
}
