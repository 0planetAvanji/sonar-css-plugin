For readability reasons, there should not be any whitespaces between ! and the important annotation.

Noncompliant Code Example

.mybox {
  width: 100px ! important;   /* Noncompliant */
  width: 100px !  important;  /* Noncompliant */
}

Compliant Solution

.mybox {
  width: 100px !important;
}

This rule is deprecated, use {rule:css:declaration-format} instead.
