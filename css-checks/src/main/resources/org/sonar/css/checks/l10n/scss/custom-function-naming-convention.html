Shared coding conventions allow teams to collaborate efficiently. This rule checks that all SCSS custom function names match a provided regular expression.

Noncompliant Code Example

With default format ^[a-z][-a-z0-9]*$: 
@function my_custom_function($color) { ... }

Compliant Solution

With default format ^[a-z][-a-z0-9]*$: 
@function my-custom-function($color) { ... }
