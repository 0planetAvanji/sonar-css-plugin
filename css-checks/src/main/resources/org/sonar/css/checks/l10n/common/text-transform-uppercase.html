For some languages, such as Greek or German, some browsers do not properly support transforming letters to uppercase. Thus, usage of the text-transform property set to uppercase or capitalize should be avoided if those languages are targeted.

Noncompliant Code Example

.mybox1 {
  text-transform: uppercase; /* Noncompliant */
}

.mybox2 {
  text-transform: capitalize; /* Noncompliant */
}

.mybox3 {
  text-transform: lowercase; /* Compliant */
}
