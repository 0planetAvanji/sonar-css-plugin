Shared coding conventions allow teams to collaborate efficiently. This rule checks that all mixin names match a provided regular expression.

Noncompliant Code Example

With default format ^[a-z][-a-z0-9]*$: 
@mixin my_mixin { ... }

Compliant Solution

With default format ^[a-z][-a-z0-9]*$: 
@mixin my-mixin { ... }
