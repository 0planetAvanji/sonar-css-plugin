For readability reasons, each declaration should be defined on a separate line.

Noncompliant Code Example

.mybox {
  background-color: blue;
  color: red; border-color: green;
}

Compliant Solution

.mybox {
  background-color: blue;
  color: red;
  border-color: green;
}
