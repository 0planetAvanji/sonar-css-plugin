Experimental column (||), descendant (>>) and deep (>>> or /deep/) combinators should not be used in production code.

Noncompliant Code Example

div >> p {
}

col.selected || td {
}

foo /deep/ p {
}

foo >>> p {
}

See

  • W3C Selectors Level 4 Specifications: Column combinator
  • W3C Selectors Level 4 Specifications: Descendant combinator
  • W3C CSS Scoping Module Level 1 Specifications: Deep combinator
