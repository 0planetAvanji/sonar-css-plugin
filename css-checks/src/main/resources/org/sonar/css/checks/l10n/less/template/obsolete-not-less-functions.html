To make sure that your code will keep working as expected in the future, do not use obsolete functions:

  • No longer supported by main vendors
  • Support to be removed by main vendors

Noncompliant Code Example

.mybox {
  font-size: min(10px, 3em); /* Noncompliant */
  font-size: max(10px, 3em); /* Noncompliant */
}
